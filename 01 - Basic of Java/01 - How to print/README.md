# How to print in Java
## print พิมพ์ในบรรทัดเดียวกัน (ไม่ขึ้นบรรทัดใหม่)
```java
System.out.print("Hello World!");
System.out.print("Hello World!");
```
## Output
```
Hello World!Hello World!
```
## println พิมพ์แล้วขึ้นบรรทัดใหม่ (ln = line)
```
System.out.println("Hello World!");
System.out.println("Hello World!");
```
## Output
```
Hello World!
Hello World!
```
